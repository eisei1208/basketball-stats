# Basketball Stats Project Overview

## 1. プロジェクト概要
バスケットボールの試合スタッツを記録・管理するWebアプリケーション。
動画を見ながらの記録を想定し、タップ操作で直感的にスタッツを入力できるUIを持つ。
Python (Flask) をバックエンド、SQLiteをデータベースとして使用。

```text
    ├── game_detail.html  # 試合詳細・スタッツ入力画面
    └── settings.html     # 設定画面
```

## 3. 機能一覧

### 試合管理
- **試合一覧表示**: 登録された試合をリスト表示。
- **新規試合作成**: 日付、対戦相手、区分、場所、勝負項目（レバレッジ）、メモを入力して試合を作成。
- **試合編集**: 対戦相手、日付、カテゴリ、勝負項目を後から編集可能。
- **試合削除**: 試合と関連するスタッツデータを削除。

### スタッツ記録
- **クオーター管理**: 1Q〜4Qは自動生成。延長(OT)はボタンで追加可能。OTの削除も可能。
- **公式スタッツ入力**: 3P, 2P, FT, リバウンド, アシスト, スティール, ブロック, ファール, ターンオーバーなどを記録。
- **専用ボーナス入力**: リングアタック、FT獲得、ルーズボール・ダイブ、ゲームメイク、データハック、キャプテン、オフェンスチャージ獲得といった「100%コントロール可能な行動」を記録。
- **入力モード切替**: ヘッダー、タブ、集計情報を非表示にし、スタッツ入力ボタンのみを表示する集中モードを搭載。
- **リアルタイム集計**: 入力に合わせて、クオーターごとの合計および試合全体の合計（得点、リバウンド、評価値）を即時更新。集計情報はタブ直下に表示。
- **試合全体タブ**: シュート成功率、各スタッツ項目・専用ボーナスの総合計を一覧表示。
- **評価値計算**: 設定された重み付けに基づいて評価ポイントを自動計算。指定した**勝負項目（レバレッジ）の評価ポイントは2倍**として計算される。

### 出場時間管理
- **タイマー**: クオーターごとの残り時間を管理（デフォルト時間は設定可能）。
- **イン/アウト記録**: 「コートイン」「コートアウト」ボタンで出場時間を記録。
- **履歴表示**: クオーターごとの出場履歴をリスト表示。削除も可能。
- **合計時間**: 試合全体での総出場時間を集計。

### 設定
- **評価ポイント設定**: 各公式スタッツ項目および専用ボーナスの重み付け（加点・減点）をカスタマイズ可能。NBA Efficiency公式設定へのリセット機能付き。
- **基本設定**: クオーターのデフォルト時間（分）を設定可能。

## 4. API エンドポイント一覧

| メソッド | エンドポイント | 説明 |
| --- | --- | --- |
| GET | `/` | 試合一覧画面を表示 |
| GET/POST | `/settings` | 設定画面の表示・更新 |
| GET/POST | `/game/new` | 新規試合作成画面の表示・作成処理 |
| GET | `/game/<int:game_id>` | 試合詳細・スタッツ入力画面を表示 |
| POST | `/api/stats/update` | スタッツの数値を更新 (+1/-1) |
| POST | `/api/game/<int:game_id>/add_ot` | 延長クオーター (OT) を追加 |
| POST | `/api/game/<int:game_id>/update` | 試合情報（対戦相手・日付・勝負項目等）を更新 |
| POST | `/api/appearance/add` | 出場時間 (IN/OUT) を記録 |
| DELETE | `/api/game/<int:game_id>` | 指定した試合を削除 |
| DELETE | `/api/quarter/<int:quarter_id>` | 指定した延長クオーターを削除 |
| DELETE | `/api/appearance/<int:appearance_id>` | 指定した出場記録を削除 |

## 5. 技術スタック
- **Backend**: Python, Flask, Flask-Migrate
- **Database**: SQLite, SQLAlchemy (Flask-SQLAlchemy)
- **Frontend**: HTML, JavaScript (Fetch API), Bootstrap 5 (CDN), Chart.js (CDN)
